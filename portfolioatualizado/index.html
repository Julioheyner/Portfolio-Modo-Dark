Sobre Mim

Graduado em Análise e Desenvolvimento de Sistemas pelo Centro Universitário FAMETRO e Desenvolvedor Backend Júnior pelo programa Oracle Next Education (ONE) T6 2024, com foco em criar aplicativos web funcionais e intuitivos que atendam às necessidades dos usuários. Tenho conhecimento em lógica de programação usando Java, C e Python, e meu principal objetivo é desenvolver soluções backend. Além disso, possuo conhecimento em frontend, utilizando HTML, CSS e JavaScript, habilidades adquiridas no início da minha trajetória como programador, também pelo programa Oracle Next Education, o que me permite integrar a parte técnica com uma experiência de usuário otimizada. Estou sempre buscando aprender e aprimorar minhas habilidades. Até o momento, desenvolvi projetos como o TextVoz, um aplicativo multilíngue que facilita a leitura de textos; o CodyLingo, um dicionário voltado para iniciantes em programação; além de ferramentas como um gerador de senha segura e um verificador da força de senha, que ajudam a promover a segurança online. Também criei uma interface de grade de ícones e um programa climático utilizando a API Open-Meteo. Meu objetivo é crescer como profissional e contribuir para projetos práticos que tenham impacto positivo. Estou sempre em busca de novas oportunidades para aprender e me manter atualizado com as tendências do setor.

Projetos

TextVoz

CodyLingo

É um dicionário online dedicado a estudantes e profissionais da área de desenvolvimento de software, cujo objetivo é fornecer um recurso abrangente para entender o vocabulário utilizado na programação. Assim como conceitos e palavras reservadas nas linguagens C e Java. Além disso, oferecemos informações detalhadas sobre as diversas profissões na área de desenvolvimento, ajudando a guiar aqueles que estão iniciando sua jornada profissional.

Badges

Challenge Decodificador

Conquistado ao completar o desafio Decodificador, focado em criptografia e codificação de dados. O Challenge Decodificador de Textos da Alura, proposto no programa Oracle Next Education, desafiou os alunos a desenvolverem uma aplicação que codifica e decodifica mensagens de texto. Utilizando JavaScript, HTML e CSS, o projeto aplicou algoritmos simples de criptografia, reforçando habilidades em manipulação de strings e lógica de programação, com uma interface intuitiva para o usuário.

Challenge Conversor

Recebido por concluir o desafio de criar um conversor de moedas. O Challenge Conversor de Moedas da turma 6 do Oracle Next Education, em parceria com a Alura, desafiou os alunos a desenvolverem uma aplicação em Java para converter moedas com base em cotações atualizadas via API Rest. O projeto reforçou o uso de APIs, manipulação de dados numéricos e implementação de lógica de conversão, oferecendo uma interface simples e eficiente para o usuário.

Challenge LiteAlura

Conquistado ao finalizar o desafio de criação de uma aplicação literária. O Challenge LiterAlura desafiou os alunos a desenvolverem uma aplicação em Java com Spring Boot para gerenciar uma biblioteca digital. A aplicação permite cadastrar, consultar, atualizar e remover livros, utilizando uma API REST e banco de dados, reforçando conceitos de backend e manipulação de dados.

Challenge Spring Boot

Recebido ao completar o desafio Spring boot, implementando uma API RESTful com o framework. O Challenge Spring Boot do programa Oracle Next Education desafiou os alunos a criarem uma aplicação backend em Java utilizando o Spring Boot. O projeto envolveu o desenvolvimento de uma API REST para gerenciar dados, com foco em boas práticas de arquitetura e organização de código. O desafio reforçou conhecimentos sobre criação de endpoints, integração com bancos de dados e testes de API, proporcionando uma experiência completa no desenvolvimento de aplicações web robustas e escaláveis.
